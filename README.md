Social Media Analytics with Number Calculator

A React application that combines social media analytics with a number calculation service.

# First_Page

<img src="Feed.png" width=200px>

## Features

- Social Media Feed
- Trending Posts
- Top Users Analytics
- Number Calculator
  - Prime Numbers
  - Fibonacci Numbers
  - Even Numbers
  - Random Numbers

## Tech Stack

- React with Vite
- Material-UI
- Axios for API calls
- React Router for navigation
