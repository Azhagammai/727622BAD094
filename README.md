## Features

- Social Media Feed
- Trending Posts
- Top Users Analytics
- Number Calculator
  - Prime Numbers
  - Fibonacci Numbers
  - Even Numbers
  - Random Numbers

## Tech Stack

- React with Vite
- Material-UI
- Axios for API calls
- React Router for navigation
